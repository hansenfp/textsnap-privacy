TextSnap

Simple tools that save time.

TextSnap is a lightweight Chrome extension that lets you extract text from anywhere on the web. Select an area, capture the content, and instantly copy clean text to your clipboard.

Built for speed. Designed for simplicity.

Privacy Policy

Effective date: 2026

1. Overview

TextSnap respects your privacy. This extension does not collect, store, or sell personal user data.

2. How the Extension Works

TextSnap allows users to select a portion of a webpage and extract text from that selected area. This action is only performed when the user actively uses the extension.

3. Data Processing

The extension processes only the content that the user explicitly selects. This may include text or images displayed on the current webpage.

4. Third-Party Services

If OCR functionality is used, selected image data may be sent to a third-party OCR service (such as ocr.space) solely for text recognition. No data is stored, tracked, or reused by the extension.

5. Data Collection

TextSnap does not collect:

  • Personal information
  • Browsing history
  • Authentication data
  • Location data

6. Data Sharing

TextSnap does not sell, share, or transfer user data to third parties, except as required to perform OCR processing when explicitly triggered by the user.

7. User Control

All functionality is initiated by the user. The extension does not run in the background or process data without user interaction.

8. Security

TextSnap is designed to minimize data exposure. No user data is stored locally or remotely by the extension.

9. Changes

This privacy policy may be updated to reflect improvements or changes in functionality.
